Un formulaire pour quoi?
- Donner la possibilité à un utilisateur de s’inscrire ;
- Donner la possibilité à un utilisateur de se connecter ;
- Permettre à un utilisateur de poster un commentaire ;
- Permettre à un utilisateur de nous envoyer un message ;
Etc..

La balise form contient toutes les autres balises du formulaire

La méthode GET (celle qui est utilisée par défaut si rien n'est renseigné) fait circuler les informations du formulaire en clair dans la barre d'adresse

La méthode POST, quant à elle, transmet les informations du formulaire de manière masquée mais non cryptée.

Attribut action: s'il n'est pas fourni, les données seront envoyées à l'URL de la page contenant le formulaire.

On récupère les données sous forme de variables:
$_GET["name"] ou $_POST["name"]
